# Example Report

This directory contains an example report (`report.md`) and all associated output files for Assignment 11.

## Contents

- `report.md` - Example written report demonstrating expected format and structure
- `output/` - All output artifacts (visualizations, CSV files, text files) referenced in the report

## About This Example

This example report was generated using **real Chicago Beach Weather Sensors data** from the official dataset:
- Source: [Beach Weather Stations - Automated Sensors](https://data.cityofchicago.org/Parks-Recreation/Beach-Weather-Stations-Automated-Sensors/k7hf-8y75/about_data)
- Dataset: 195,672 records from April 2015 to November 2025
- Three weather stations: 63rd Street, Foster, and Oak Street
